# backend 这个文件夹是做啥用的

- 这是“后端服务”的代码目录（Back-end API）。
- 用 Node.js + Express 提供接口给前端使用。
- 默认端口：63002（前端的 /api 请求会被代理到这里）。
- 快速启动：

```powershell
cd .\backend
npm install
node simple-server.js   # 如需改端口：$env:PORT=63001; node simple-server.js
```

- 常用地址：
  - 健康检查: http://127.0.0.1:63002/health
  - 示例接口: http://127.0.0.1:63002/api/surveys

- 想换端口：运行前设置环境变量 `PORT`（示例见上面注释）。
