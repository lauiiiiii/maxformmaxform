## 新确认的问题

### 1. smoke 入口并行限制已解除

今天在完成脚本改造后并行执行：

- `node scripts/system-smoke.mjs`
- `cmd /c npm run smoke:system`

两套入口均通过。当前策略是：未显式设置 `PORT` 时由系统分配空闲端口，默认按 `DB_NAME + runId` 创建独立临时库并在结束后清理，因此并行执行不会再因为固定端口或共享测试库互相影响。
