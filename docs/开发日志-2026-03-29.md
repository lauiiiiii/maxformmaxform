## 新确认的问题

### 1. smoke 入口仍不能并行

今天并行执行：

- `node scripts/system-smoke.mjs`
- `cmd /c npm run smoke:system`

实际触发了 `63102` 端口冲突，后端启动报 `EADDRINUSE`，前端 npm 入口返回 `TypeError: fetch failed`。因此文档里继续保留“顺序单实例执行”的限制是必要的，而且这次已经有了 `2026-03-29` 的实测依据。
